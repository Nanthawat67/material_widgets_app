# Material Widgets and Navigation Application

แอปพลิเคชันนี้จัดทำขึ้นเพื่อศึกษาและฝึกปฏิบัติการใช้งาน  
Material Component Widgets และ Navigation and Routing ของ Flutter  
ตามรายวิชา Mobile Devices Application Design and Development

## วัตถุประสงค์
- เพื่อศึกษาการใช้งาน Material Widgets ในแต่ละหมวด
- เพื่อฝึกการใช้งาน Navigation และ Routing ใน Flutter
- เพื่อแสดงโค้ดตัวอย่างพร้อมผลลัพธ์บนหน้าจอ

## โครงสร้างแอป
แอปประกอบด้วยหน้าจอหลัก และหน้าจอย่อย 6 หน้า ได้แก่
- Actions
- Communication
- Containment
- Navigation
- Selection
- Text Inputs

## Navigation and Routing
แอปนี้ใช้ Named Routes ในการจัดการเส้นทาง  
โดยกำหนด route name ให้แต่ละหน้า และเรียกใช้งานผ่าน  
Navigator.pushNamed ซึ่งช่วยให้โค้ดมีความเป็นระเบียบ  
และเหมาะสำหรับแอปที่มีหลายหน้าจอ

## สรุป
จากการพัฒนาแอปนี้ทำให้เข้าใจการทำงานของ Material Widgets  
และ Navigation ใน Flutter มากยิ่งขึ้น  
สามารถนำความรู้ไปประยุกต์ใช้ในการพัฒนาแอปพลิเคชันจริงได้
